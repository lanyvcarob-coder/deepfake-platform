#  Plataforma de Detección de Deepfakes

##  Descripción
Este proyecto consiste en el desarrollo de una plataforma web para la detección de deepfakes, implementada bajo la metodología Extreme Programming (XP).

El sistema permite:
- Registro e inicio de sesión de usuarios
- Control de acceso basado en roles
- Carga de archivos multimedia (imágenes y videos)
- Análisis de contenido mediante un modelo de detección de deepfakes (simulado)
- Generación de reportes con resultados del análisis

---

##  Tecnologías utilizadas
- Python
- Flask
- SQLite
- JWT (JSON Web Tokens)
- bcrypt
